 ## Step to Start Project
 

 
**For Backend**
 1. `git clone https://github.com/MohitSojitra/e-commerce-store.git`
 2. `cd e-comerce-backend`
 3. `npm i`
 4. `npm run data:import`
 5. `npm run dev`

**For Frontend**

 1. `cd e-comerce-frontend`
 2. `npm i`
 3. `npm run start`
